# Task-3--Dashboard-Design
## Objective 
Design an interactive dashboard for business stakeholders using Power BI, so they can quickly understand sales performance, profitability, and trends without needing a new report for every question.
## Tools Used
Microsoft Power BI Desktop
Powerpoint for summary 
## Dataset 
Superstore dataset
## What I did 
1. Created DAX measures for Total Sales, Total Profit, Total Margin and Order Count
2. Built four KPIs card visuals, each colour coded and formatted for readbility
3. Added 3 slicers for interactivity(Region, Category, and Order Date)
4. Added a time-series line cahrt plotting Sales(ansd Profit) trends over Order Date
## Key insights 
- Sales and profit can be filtered instantly by region, category and order date
- Profit margin sits around 12-17%, giving a quick health chevk on pricing/cost efficiency
- The time-series view makes it easy to spot seasonal peaks and dips in sales
## Deliverables in this repo 
- Task3_Dashboard.pbix - the interactive Power BI dashboard file
- Task3_Dahboard_Summary.pptx - PPT summary of project
## Outcome 
Learned how to design a stakeholder-raedy, interactive Power BI dashboard that turns raw sales data into decision-ready insights using KPIs, slicers, and time-series analysis.
